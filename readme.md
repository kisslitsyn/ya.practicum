#### 2. credit_score

###### Описание: 
Набор данных о клиентах банка: возраст, трудовой стаж, семья, дети, наличие задолженности и др. Необходимо выяснить, влияют ли отдельные характеристики клиента на факт погашения кредита в срок.

Исследование надежности заемщиков. Предобработка данных
зультаты исследования будут учтены при построении модели кредитного скоринга — специальной системы, которая оценивает способность потенциального заёмщика вернуть кредит банку.
